friend groups

what is this?

This is a tool that made for the public school I work at, to create chaperone groups for field trips based on students' friend groups. Sometimes over 100 kids will go on these trips, each putting down 5 or 6 friends that they want in their group. This is slightly difficult to do by hand.
Export student responses as a CSV file, then use the Choose File button below to upload your spreadsheet. The tool will create groups based on the students' friend groups.
Most students will put down friends of the same gender, so to make more cohesive friend groups, please prefix the first name in each row with the token for boy or girl below (I'll make this step optional Eventually™).
